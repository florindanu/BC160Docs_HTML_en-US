Special codes for Company information, Customers, and Vendors

The Russian version of Business Central adds special classified codes that are needed in some printing forms and reports.

Company Information

In the Company Information page, you can specify the codes:

Fast Tab General:

  • VAT registration No

Fast Tab Payments:

  • KPP Code
  • Bank BIC

Fast Tab Reporting:

  • Pension Fund Registration No
  • Medical Fund Registration No
  • Social Insurance Code
  • Tax Bearer Canegory Code
  • OGRN Code
  • OKPO Code
  • OKONX Code
  • OKOPF Code
  • OKFS Code
  • OKOGU Code
  • OKATO Code
  • OKVED Code
  • Recipient Tax Authority SONO
  • Admin. Tax Authority SONO

Vendors

In the Vendor Card page, the following fields are added:

Fast Tab Invoicing

  • Vat Registration No
  • OKPO Code
  • OKONX Code
  • KPP Code

Customers

In the Customer Card page, the following fields are added:

Fast Tab Invoicing:

  • VAT Registration No

FastTab Payments:

  • KPP Code
  • OKPO Code
