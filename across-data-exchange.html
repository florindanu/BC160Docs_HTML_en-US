Exchanging Data Electronically

You can use the Data Exchange Framework to manage the exchange of business documents, bank files, currency exchange rates, and any other data files with your business partners.

In the standard version of Business Central, the Data Exchange Framework is used in features, such as Electronic Documents, Bank File Import/Export, and Currency Exchange Rates Update. For more information, see About the Data Exchange Framework.

As an administrator or Microsoft partner, you can use the framework in new integration features by setting up which data to exchange and how. For more information, see Set Up Data Exchange Definitions.
